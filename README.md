# RSVP Movies SQL Data Analysis

## Project Overview
This project analyzes the RSVP Movies dataset using SQL to generate business insights.

The analysis focuses on:
- Movie ratings
- Genre performance
- Production companies
- Language trends
- Business insights from movie data

## Tools Used
- SQL
- Git & GitHub

## Key Analysis Performed
- Top rated movies
- Genre-wise analysis
- Production company performance
- Actor and director analysis
- Rating trends

## SQL Skills Demonstrated
- Joins
- Aggregations
- Window Functions
- Subqueries
- CTEs
